### Hardware

- BIOS Version: P4.40  2020-01-17
- Motherboard: ASRock H310M-STX/COM
- CPU: Intel 9th i7-9700
- iGPU: Intel UHD 630
- Memo: ZHIKE 32G（16G*2）DDR4 2666 Mhz

- HDA: Realtek ALC233
- LAN: Intel L219-V
- WiFI: BCM94360CS2
